Project Data

The Claremont Run tracks quantitative data on Chris Claremont's Uncanny X-Men run. Data currently tracked in the project includes (but is not limited to):

  • Character visualizations
  • Character speech
  • Bechdel test pass/fails per issue
  • Character behaviour/actions
  • Issue covers, including character visualizations and narrations
  • Locations within each issue

The raw spreadsheets for each dataset can be viewed and downloaded below:
  • Character behaviour, Bechdel test results, cover logging, and issue setting tracking spreadsheets: character_and_location_data.xlsx
  • Character visualizations and speech: character_visualization_data.xlsx
